Questions de Cours
HTML & CSS

Questions Réponses
1. Donner la structure de base d'un document HTML5. La structure de base est :
<!DOCTYPE html>
<html lang="fr">
<head>
<meta charset="UTF-8">
<title>Titre</title>
</head>
<body>
<!-- Contenu -->
</body>
</html>
2. À quoi sert la propriété CSS display: flex ? display: flex active le modèle Flexbox sur un conteneur. Il permet de :
• Aligner les éléments enfants horizontalement ou verticalement
• Distribuer l'espace entre les éléments
• Créer des mises en page souples sans recourir aux float
3. Citez deux balises HTML utilisées dans les formulaires. • <input> : permet de créer des champs de saisie (texte, mot de passe, case à cocher, etc.)
• <button> : crée un bouton cliquable (soumission, réinitialisation, etc.)
Autres exemples : <textarea>, <select>, <label>
4. Que produit le code suivant : <h1 style="color:red">Bienvenue</h1> Ce code affiche le texte « Bienvenue » en rouge, mis en forme comme un titre de niveau 1 (grand et gras par défaut). L'attribut style applique directement la propriété CSS color:red en ligne (inline style).
5. Expliquez la différence entre id et class. id : identifiant unique dans la page. Un seul élément peut avoir un id donné. En CSS : #monId
class : peut être partagée par plusieurs éléments. En CSS : .maClasse
6. Quelle propriété CSS permet de : a) changer la couleur du texte ? b) changer la couleur de fond ? a) color : change la couleur du texte
b) background-color : change la couleur de fond
7. À quoi sert l'attribut href dans une balise <a> ? L'attribut href (Hypertext REFerence) indique la destination du lien. Il peut contenir :
• Une URL absolue : https://www.exemple.com
• Une URL relative : page2.html
• Une ancre : #section1
• Un email : mailto:contact@exemple.com
8. Quelle est la différence entre une balise bloc (block) et une balise en ligne (inline) ? Bloc (block) : occupe toute la largeur disponible et commence sur une nouvelle ligne. Exemples : <div>, <p>, <h1>
Inline : n'occupe que l'espace nécessaire et reste dans le flux du texte. Exemples : <span>, <a>, <strong>
9. Quel est le rôle des balises : <html>, <head>, <body> ? <html> : élément racine qui englobe tout le document HTML
<head> : contient les métadonnées (titre, encodage, liens CSS, scripts)
<body> : contient tout le contenu visible de la page
10. Définissez les termes : HTML et CSS HTML : HyperText Markup Language - langage de balisage utilisé pour structurer le contenu d'une page web
CSS : Cascading Style Sheets - langage de feuilles de style utilisé pour mettre en forme et présenter visuellement les éléments HTML
