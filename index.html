Question

Qu'est-ce que le développement web ? Expliquez la différence entre le HTML, le CSS et le JavaScript, en donnant un exemple concret du rôle de chacun dans la création d'une page web.

Éléments de réponse attendus

Le développement web désigne l'ensemble des activités liées à la création et à la maintenance de sites et d'applications accessibles via un navigateur internet. Il repose sur trois technologies fondamentales complémentaires :

  • HTML - structure et contenu de la page (titres, paragraphes, images, liens). C'est le squelette du site.
  • CSS - présentation visuelle (couleurs, polices, mise en page). C'est l'habillage du site.
